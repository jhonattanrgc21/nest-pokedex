# Ejecutar en desarrollo

1. Clonar el repositorio
2. Tener Nest CLI instalado
```
npm i -g @nest/cli
```
3. Ejecutar
```
npm i
```


4. Levantar la Base de Datos de docker
```
docker-compose up -d
```

## Stack utilizado
* MongoDB
* Nest.js
